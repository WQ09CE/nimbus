# Web UI 增强功能文档

## 概述
本次重构主要针对 web-ui 的消息显示系统进行了大幅增强，包括更好的 Markdown 支持、智能数据显示以及改进的UI布局。

## 主要改进

### 1. 消息显示顺序优化 ✅
- **Tool Calls 在最上方**：所有工具调用及其结果显示在消息的顶部
- **最终回复在最下方**：AI的最终回复内容显示在所有工具调用之后
- **视觉分隔**：最终回复有清晰的视觉标识和分隔线

### 2. 增强的 Markdown 渲染器
#### 新增支持的格式：
- **扩展标题**：支持 H1-H4，带有边框装饰
- **删除线**：~~文本~~ 格式支持
- **高亮文本**：==文本== 格式支持
- **任务列表**：- [ ] 和 - [x] 复选框支持
- **表格**：完整的 Markdown 表格支持，带有悬停效果
- **图片**：![alt](url) 图片显示支持
- **键盘按键**：自动检测 Ctrl+C 等按键组合
- **代码复制**：代码块自带复制按钮

#### 增强功能：
- **代码块语言标识**：右上角显示编程语言
- **一键复制代码**：点击📋按钮复制代码
- **链接增强**：链接显示🔗图标
- **响应式设计**：移动端优化

### 3. 智能数据显示系统 (DataDisplay)
#### 智能类型检测：
- **URL 检测**：自动识别链接，显示🔗图标
- **文件路径**：识别文件路径，显示📁图标
- **邮箱地址**：识别邮箱，显示📧图标，可点击发送
- **JSON 数据**：自动解析和格式化 JSON 字符串
- **Base64 数据**：识别 Base64 编码，提供复制功能
- **长文本**：超过300字符的文本提供展开/收起功能

#### 结构化数据显示：
- **数组**：智能列表显示，相同类型数据紧凑显示
- **对象**：层级显示，简单对象紧凑模式
- **特殊类型**：Date、Error 等特殊对象的专门显示
- **数值**：大数值显示千分位分隔符和科学计数法
- **布尔值**：✅❌ 图标显示
- **深度控制**：防止过深嵌套，可配置最大深度

### 4. 视觉和交互改进
#### 视觉增强：
- **类型图标**：不同数据类型有对应的表情图标
- **渐变背景**：最终消息区域有微妙的渐变背景
- **状态指示器**：改进的加载动画和状态显示
- **悬停效果**：交互元素的悬停反馈

#### 交互改进：
- **展开/收起**：大型数据结构可折叠
- **复制功能**：代码和长文本一键复制
- **响应式布局**：移动端适配

### 5. 性能优化
- **懒加载**：图片懒加载
- **虚拟化**：大型数据集合的性能优化
- **缓存机制**：Markdown 渲染结果缓存

## 使用示例

### Markdown 支持示例
```markdown
# 主标题
## 子标题
**粗体文本** *斜体文本* ~~删除线~~ ==高亮==

- [ ] 待办事项
- [x] 已完成项目

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |

```javascript
function hello() {
  console.log("Hello World");
}
\`\`\`

![图片](https://example.com/image.jpg)
```

### 数据显示示例
- **URL**：`https://example.com` → 🔗 链接
- **文件路径**：`/home/user/file.txt` → 📁 路径
- **JSON**：自动格式化显示
- **数组**：`[1, 2, 3]` → 紧凑显示
- **对象**：层级树状显示

## 配置选项

### MarkdownRenderer 配置
```typescript
<MarkdownRenderer 
  content={content}
  className="custom-styles"
/>
```

### DataDisplay 配置
```typescript
<DataDisplay 
  data={data}
  title="数据标题"
  maxDepth={3}           // 最大显示深度
  currentDepth={0}       // 当前深度
/>
```
